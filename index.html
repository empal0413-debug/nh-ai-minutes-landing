녹음시작 → 자동업로드 → 회의록

권한 허용 후 녹음을 시작하면 업로드까지 자동 연결되며, 실패 파일은 최근회의록에서 재전송할 수 있습니다.

+ 🎙️ 녹음시작 버튼으로 즉시 시작

-

사용자가 얻는 3가지 이점

-

실무에서 가장 번거로운 회의록 작성/공람/보관 프로세스를 자동화합니다.

- -
-
-
🎙️
-

자동 텍스트 변환 + 화자분리

-

앱에서 녹음하면 음성을 자동으로 텍스트로 변환하고, 화자를 분리해 회의 내용을 정확히 정리합니다. 이후 회의록 자동요약 생성과 공람까지 연결됩니다.

+
+

핵심 이점

+

회의 중 조작 실수는 줄이고, 종료 후 후처리 시간을 단축합니다.

+
+
+

1) 녹음 중심 UI

+

녹음/일시정지/종료 상태를 크게 보여 회의 중 빠르게 제어할 수 있습니다.

- -
-
📧
-

전자우편 공유로 공람 간소화

-

참석자들에게 전자우편으로 회의록을 공유해 복잡한 공람 절차를 줄이고, 빠른 확인과 협업을 지원합니다.

+
+

2) 실패 대응 동선

+

업로드 실패 파일 노출과 재전송 버튼으로 누락 위험을 낮춥니다.

- -
-
🗂️
-

서버 자동보관 + 유연한 열람 공유

-

회의록은 서버에 자동 보관되어 참석자가 언제든 열람할 수 있으며, 필요 시 제3자를 열람자로 추가해 공유할 수 있습니다.

+
+

3) 자동 처리 연결

+

녹음 종료 직후 업로드와 회의록 생성 단계로 자연스럽게 이어집니다.

-
-

접속/설치 위치 안내

-

웹과 앱 어디서든 바로 시작할 수 있도록 경로를 안내합니다.

+
+

앱 화면 기준 사용 흐름

+

실제 운영에서 자주 발생하는 권한/업로드/종료 예외까지 포함했습니다.

+
+
STEP 01스플래시/접속중

앱 실행 시 로고와 접속 상태 표시

+

초기 사용자도 바로 시작 가능

+
STEP 05권한 미허용 예외

상세설정 이동으로 즉시 복구

+
STEP 06최근회의록/실패목록

+

토스트 메시지로 성공 확인

현재 상태를 명확하게 표시
